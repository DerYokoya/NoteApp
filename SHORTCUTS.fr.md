## Raccourcis clavier

### Opérations sur les fichiers
| Raccourci | Action |
|----------|--------|
| `Ctrl + N` | Nouvel onglet |
| `Ctrl + O` | Ouvrir un fichier |
| `Ctrl + S` | Enregistrer |
| `Ctrl + Maj + S` | Enregistrer sous |
| `Ctrl + W` | Fermer l'onglet |
| `Ctrl + Suppr` | Supprimer le fichier |
| `Ctrl + Q` | Quitter l'application |
| `Ctrl + P` | Imprimer le document |
| `Ctrl + Maj + E` | Exporter au format PDF |

### Opérations d'édition
| Raccourci | Action |
|----------|--------|
| `Ctrl + Z` | Annuler |
| `Ctrl + Y` | Rétablir |
| `Ctrl + F` | Rechercher |
| `Ctrl + H` | Rechercher et remplacer |
| `F3` | Rechercher suivant |
| `Maj + F3` | Rechercher précédent |
| `Échap` | Fermer la barre de recherche |

### Mise en forme du texte
| Raccourci | Action |
|-----------|--------|
| `Ctrl + B` | Gras |
| `Ctrl + I` | Italique |
| `Ctrl + U` | Souligné |
| `Ctrl + K` | Insérer/modifier un lien |
| `Ctrl + \` | Effacer la mise en forme |
| `Ctrl + Maj + P` | Exposant |
| `Ctrl + Maj + B` | Indice |
| ``Ctrl + ` `` | Insérer un bloc de code |

### Alignement
| Raccourci | Action |
|----------|--------|
| `Ctrl + L` | Aligner à gauche |
| `Ctrl + E` | Centrer |
| `Ctrl + R` | Aligner à droite |
| `Ctrl + J` | Justifier |

### Listes et tableaux
| Raccourci | Action |
|----------|--------|
| `Ctrl + Maj + L` | Liste à puces |
| `Ctrl + Maj + N` | Liste numérotée |
| `Ctrl + T` | Insérer un tableau |
| `Ctrl + Maj + T` | Propriétés du tableau |
| `Tab` | Augmenter l'indentation |
| `Maj + Tab` | Diminuer l'indentation |

### Navigation entre les onglets
| Raccourci | Action |
|----------|--------|
| `Ctrl + 1` à `Ctrl + 8` | Passer à l'onglet 1-8 |
| `Ctrl + 9` | Passer au dernier onglet |

### Affichage
| Raccourci | Action |
|----------|--------|
| `Ctrl + Maj + D` | Basculer entre le thème clair et le thème sombre |

### Interligne
| Raccourci | Action |
|----------|--------|
| (Via le menu déroulant) | Simple |
| (Via le menu déroulant) | 1,5 ligne |
| (Via le menu déroulant) | Double |

### Images
| Raccourci | Action |
|----------|--------|
| (Via le bouton de la barre d'outils) | Insérer une image |
| (Clic droit) | Redimensionner l'image |

### Fonctionnalités supplémentaires
| Fonctionnalité | Accès |
|---------|--------|
| Ligne horizontale | Bouton de la barre d'outils |
| Couleur du texte | Sélecteur de couleur de la barre d'outils |
| Couleur de surlignage | Sélecteur de couleur de la barre d'outils |
| Famille de polices | Menu déroulant des polices |
| Taille de la police | Menu déroulant des tailles |
| Styles de texte | Menu déroulant des styles (Normal, Titre, Titres 1 à 6, Bloc de code) |

### Actions du menu contextuel
- Opérations sur les tableaux (Insérer/Supprimer des lignes/colonnes)
- Propriétés du tableau
- Options d'alignement
- Redimensionnement d'image (en cliquant avec le bouton droit sur une image)
